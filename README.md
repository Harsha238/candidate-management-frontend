## Description
The Candidate Management System is a full stack web application designed to manage candidate information efficiently. It allows users to add, view, and delete candidate records through an interactive and responsive user interface.

The frontend is built using React, providing a dynamic and user-friendly experience. The backend is developed using Flask, which exposes REST APIs to handle candidate operations. SQLite is used as the database to store candidate details permanently.

## Features
- Add candidate
- View candidate list
- Delete candidate
- Full stack integration
- Responsive UI

## Tech Stack
Frontend: React, HTML, CSS, JavaScript  
Backend: Python, Flask  
Database: SQLite  
Deployment: Netlify, Render  
Version Control: Git, GitHub
